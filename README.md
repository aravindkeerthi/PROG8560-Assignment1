# PROG8560-Assignment1

## Prerequisites

Visual Studio Code, NodeJS and Express JS

## Steps

### 1. Clone the source code
```sh
$ git clone https://github.com/aravindkeerthi/PROG8560-Assignment1.git
$ cd PROG8560-Assignment1
```
### 2. Install Mongoose
```sh
$ npm install mongoose
```
### 3. Install nodemon
```sh
$ npm -g nodemon
```
### 4. Run the project
```sh
$ npm run dev
